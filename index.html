Publish Sphinx Docs Documentation¶

Introduction¶

A simple Python 3.6+ CLI to publish your Sphinx documentation to a Github Pages or Gitlab Pages repository.

Installation¶

pip install publish_sphinx_docs

Usage¶

At the command line, enter publish-sphinx-docs --help.

Authors¶

  • Alex Raichev, 2018-12-03

Module main¶

publish-sphinx-docs¶

Use Sphinx, which needs to be installed in your virtual environment, to build your documentation as a single HTML page, then push the result to your Github Pages or Gitlab Pages repository.

publish-sphinx-docs [OPTIONS]

Options

--docs-dir <docs_dir>¶

Name of your Sphinx documentation directory [default: docs]

--docs-remote <docs_remote>¶

URL of your Github Pages or Gitlab Pages repository e.g. https://github.com/user/repo or git@github.com:user/repo [default: <function <lambda> at 0x7fd77ea29b70>]
